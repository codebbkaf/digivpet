# Digimon Color and Pendulum Color Complete Evolution Tree

This document contains the complete evolution tree map for both the **Digital Monster Color (1~5)** and **Digimon Pendulum Color (0~5)** devices. All trees use standard English stage names and directional arrows for progression.

---

## Part 1: Digital Monster Color (Versions 1~5)

### Version 1 (Original Brown / Gray)
* Fresh: Botamon -> In-Training: Koromon
    * Rookie: Agumon
        * Champion: Greymon / Tyranomon / Devimon / Meramon / Numemon
    * Rookie: Betamon
        * Champion: Devimon / Meramon / Airdramon / Seadramon / Numemon
* Champion to Ultimate (Perfect):
    * Greymon / Devimon / Airdramon -> Ultimate: MetalGreymon (Virus)
    * Tyranomon / Meramon / Seadramon -> Ultimate: Mamemon
    * Numemon -> Ultimate: Monzaemon
* Ultimate to Mega:
    * MetalGreymon (Virus) -> Mega: BlitzGreymon
    * Mamemon -> Mega: BanchoMamemon
    * Monzaemon -> Mega: ShinMonzaemon
* Ultra (Jogress):
    * BlitzGreymon -> Ultra: Omegamon Alter-S (Jogress with CresGarurumon)

    * Vademon -> Mega: Ebemon
* Ultra (Jogress):
    * CresGarurumon -> Ultra: Omegamon Alter-S (Jogress with BlitzGreymon)
